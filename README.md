# ManageDle

A centralized hub for managing your daily Wordle-style games. Track completions, record scores, and maintain your gaming streaks all in one place.

## Features

- **Game Roster**: Curate your personal list of daily word games
- **Score Tracking**: Manually log your daily results and statistics
- **Statistics Dashboard**: View streaks, win rates, and score distributions
- **Local Storage**: All data stored locally (cloud sync coming soon)
- **Future Features**: Social comparison, automatic score tracking, friend leaderboards

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Zustand (state management)
- Local Storage (data persistence)
