# ManageDle Change Log

## [V1.0.0] - 2026-01-04
A daily puzzle hub to track, manage, and compare your daily puzzle games.

### Features
- **Dashboard** — View and manage your active daily games in a responsive grid
- **Game Cards** — See scores, emoji grids, reset timers, and completion status at a glance
- **Organize Mode** — Drag-and-drop reordering with auto-scroll, fullscreen overlay for focused reorganization
- **Todo Sort** — Temporarily sort incomplete games to the top
- **Score Entry** — Log scores manually or paste share text; supports multi-puzzle games (e.g., LoLdle)
- **Stats Modal** — View per-game statistics and historical records
- **Leaderboard** — Compare scores with other users; filter by week, month, or year
- **User Profiles** — Display play streaks, win streaks, max win streaks, and game stats
- **Authentication** — Sign up, log in, and manage your account (Supabase-backed)
- **Theme Toggle** — Switch between dark and light modes
- **Tooltips** — Hover over game icons to see descriptions from the database
- **Changelog Modal** — View the changelog directly in the app
- **Feedback / Tickets** — Submit feedback or report issues from the header
- **Add / Remove Games** — Add games from the sidebar or remove from your dailies with confirmation
- **Clear All** — Remove all games from active dailies with confirmation prompt
- **Responsive UI** — Works on desktop and mobile browsers
- **Supabase Integration** — Cloud sync for games, records, and user data
